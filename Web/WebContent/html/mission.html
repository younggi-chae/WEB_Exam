html이란?

HTML 은 Hyper Text Markup Language 약어로 HyperText(웹 페이지에서 다른 페이지로 이동할 수 있도록 하는 것) 기능을 가진 문서를 만드는 언어입니다. 다시 말해, 구조를 설계할 때 사용되는 언어로 hyper link 시스템을 가지고 있으며, 흔히 말하는 웹 페이지를 위한 마크업 언어라고 할 수 있습니다.

html 구성요소

doctype
브라우저는 현재 문서가 어떠한 형태의 문서인지를 판단하는 목적으로 DOCTYPE을 확인합니다.
head
브라우저에게 HTML 문서의 머리부분임을 알려줍니다.
body
웹브라우저에 보여질 문서의 내용을 작성하는 부분 입니다.

html 태그

html 태그
태그 태그설명
h1~6 태그를 이용하여 제목을 표시합니다.
br 태그를 이용하여 줄 바꿈을 합니다.
strong 태그를 이용하여 문자를 강조합니다.
a 하나의 문서에서 다른 문서로 연결하기 위해 (하이퍼링크) 사용합니다.
p paragraph, 즉 문단의 약자로, 하나의 문단을 만들 때 쓰입니다.
div Division의 약자로, 레이아웃을 나누는데 주로 쓰입니다.
html 목록태그
태그 태그설명
ul 번호 없는 목록을 표시
ol 번호 있는 목록을 표시
li ul, ol 태그 하위에서 각 항목을 나타냄
dl 용어 정의와 설명에 대한 내용을 목록화해서 표시
dt 용어 목록의 정의 부분을 나타냄
dd 용어 목록의 설명 부분을 나타냄
